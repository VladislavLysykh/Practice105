# AI Text Helper

AI Text Helper — это микросервисное веб-приложение для обработки и улучшения текста.

Пользователь вводит текст, выбирает режим обработки, после чего приложение создаёт задачу. Длительная обработка выполняется отдельным worker-сервисом, а backend только принимает запросы и управляет статусами задач.

Проект разработан для практической работы по теме Docker Compose и микросервисной архитектуры.

## Возможности приложения

Приложение поддерживает несколько режимов обработки текста:

- улучшение текста;
- исправление ошибок;
- перевод текста в официальный стиль;
- перевод текста в дружелюбный стиль;
- сокращение текста.

## Архитектура проекта

Приложение состоит из нескольких сервисов:

| Сервис | Назначение |
|---|---|
| nginx | Единая внешняя точка входа |
| frontend | Пользовательский интерфейс |
| backend | WSGI API для обработки HTTP-запросов |
| worker | Отдельный сервис для выполнения длительных задач |
| redis | Очередь задач и хранение результатов |

## Схема взаимодействия сервисов

```text
Пользователь
    |
    v
Nginx
    |
    |----> Frontend
    |
    |----> Backend API
                |
                v
              Redis
                |
                v
              Worker

```

Пользователь отправляет запрос в приложение через Nginx.

Nginx является единой внешней точкой входа и перенаправляет запросы:

- запросы к интерфейсу отправляются во frontend;
- запросы по пути `/api/` отправляются в backend.

Backend принимает запрос, создаёт задачу и помещает её в Redis.

Worker получает задачу из Redis, выполняет длительную обработку текста и сохраняет результат.

Итоговая схема:

Пользователь → Nginx → Frontend  
Пользователь → Nginx → Backend API → Redis → Worker

## Инструкция по запуску

1. Установить Docker и Docker Compose.

2. Перейти в корень проекта, где находится файл `docker-compose.yml`:

```bash
cd ErmolinskayaAA
```
3. Запустить проект командой:
docker compose up --build

4. После запуска открыть приложение в браузере:
  http://localhost
